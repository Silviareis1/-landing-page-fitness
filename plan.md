# Projeto de Landing Page para App fitness

Página com fundo branco e fonte Roboto como principal.
Objetivo é ter uma landing page clean, branca e sofisticada.

## Seção HEADER:

Logo no canto esquerdo
- A logo é um SR.Dev em um círculo (feito com CSS)
- o fundo do círculo é um gradiente com: 70% de roxo e 30% de azul ciano

Botão de download no canto direito
- fundo roxo, cor branca
- borda arredondada

## Seção HERO:

A hero deve ter altura 90vh, divido em 3 áreas horizontais:

Esquerda:
Apenas a headline, sem ser grande demais, sem destaque (o destaque é o print do app no meio).
"O app perfeito para acompanhar sua vida fitness."

Centro:
Em destaque a imagem images/print_1.png que é um print do app.

Direita:
Uma lista de pontos-chave sobre o app:
- ❤️ Saúde
- 🏃 Exercício
- 🤝 Amigos

## Seção de FUNCIONALIDADES:

Vão ser usadas as imagens images/print_2.png, images/print_3.png e images/print_4.png que serão exibidas da seguinte forma:
- print / texto
- texto / print
- print / texto

Ou seja, irá alternar entre o print na esquerda e um texto explicativo na direita, vice-versa.

Descrição breve de cada print para você gerar um texto adequado:
- print_2.png = Contagem de calorias
- print_3.png = Exercícios específicos (corrida, musculação, etc)
- print_4.png = Compartilhamento com amigos

## Seção de CTA (call-to-action):
Botão grande de download com um texto abaixo dele destacando:
"Totalmente gratuito"
